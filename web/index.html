「계산은 정답이 있고,
해석은 정답이 없다」

— 이 플랫폼을 관통하는 한 문장

分

계산과 해석의 분리

팔자·대운·괘는 결정론으로 산출하고 천문 oracle 로 교차검증합니다. 철학자 랭킹도 그래프 회수의 결정론 집계 — LLM 서술이 계산을 오염시키지 않습니다.

據

모든 풀이에 근거

사주 리포트의 모든 문장은 출처(trace)를 동반하고, 리포트 아래 「이 풀이의 근거」 푸터로 사용된 값을 공개합니다. 철학 진단은 SEP 원문 인용을 근거로 답합니다.

竝

관점의 병렬

하나의 결론을 강요하지 않습니다. 유파별 · 사상별 관점을 나란히 놓고, 합의되는 부분과 갈리는 부분을 먼저 드러냅니다.
