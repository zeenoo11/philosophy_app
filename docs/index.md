# 철학적 4대 지표 가이드

## 1. Agency (주체성)
**당신은 운명의 작가인가요, 아니면 배역을 맡은 배우인가요?**

*   **능동 (1.0)**: "내 인생은 내가 만든다." 외부 환경보다 자신의 의지와 선택이 삶을 결정한다고 믿습니다 (예: 실존주의, 인본주의).
*   **수동 (0.0)**: "삶은 거대한 흐름이다." 운명, 신의 뜻, 또는 거스를 수 없는 자연의 섭리에 순응하는 태도입니다 (예: 결정론, 도가 사상).

## 2. Logic (판단 근거)
**세상을 이해하는 당신의 렌즈는 무엇인가요?**

*   **이성 (1.0)**: "데이터와 논리가 진리다." 객관적인 사실, 과학적 증거, 합리적인 추론을 중시합니다 (예: 스토아 철학, 현실주의, 실용주의).
*   **감성 (0.0)**: "직관과 마음이 진실이다." 주관적인 경험, 감정의 울림, 예술적 영감을 더 중요하게 여깁니다 (예: 낭만주의, 에피쿠로스).

## 3. Focus (관심 반경)
**당신의 에너지는 어디를 향하고 있나요?**

*   **개인 (1.0)**: "나의 자유와 성장이 먼저다." 개인의 독립성, 자아 실현, 내면의 평화를 추구합니다 (예: 결단주의, 개인주의).
*   **공동체 (0.0)**: "우리는 연결되어 있다." 타인과의 관계, 사회적 책임, 조화와 연대를 중시합니다 (예: 유교, 공리주의).

## 4. Outlook (삶의 태도)
**당신은 내일의 태양을 어떻게 바라보나요?**

*   **낙관 (1.0)**: "삶은 살만한 가치가 있다." 미래에 대한 희망, 긍정적인 가능성, 인간의 선함을 믿습니다 (예: 인본주의, 이상주의).
*   **비관 (0.0)**: "삶은 고통이거나 무의미하다." 현실의 부조리함, 인간의 한계, 냉혹한 진실을 직시합니다 (예: 염세주의, 허무주의).
